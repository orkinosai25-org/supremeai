### Logo Usage

- Use **`logo-dark.svg`** on dark or coloured backgrounds.
- Use **`logo-light.svg`** on white or light backgrounds.
- Use **`icon.svg`** wherever a square/circular icon is needed.
- Maintain clear space equal to the crown mark height on all sides.
- Minimum wordmark width: **160 px**.

### Typography

- **Headings:** Inter 700 / 800
- **Body:** Inter 400 / 500
- **Code:** JetBrains Mono 400 / 500

## Spacing & Sizing

- Minimum logo width: **120px** (dark/light variants)
- Minimum mascot size: **48px**
- Clear-space around logo: **½ × logo height** on all sides
- Do not stretch, rotate, or recolor logos outside the approved palette

## Design Principles

1. **Regal & Luxurious** — Crown motif signals premium quality; gold and silver reinforce authority.
2. **Gem-lit** — Each letter/element carries a gemstone metaphor: Emerald **S**, Diamond **ai**, Rainbow letters.
3. **Cosmic-Tech** — Dark starfield backgrounds blend sci-fi tech (circuit board crown) with luxury.
4. **Playful Intelligence** — The colorful brain/planet mascot brings warmth and accessibility.
5. **Spectrum of AI** — Rainbow gradients reference the diverse range of AI models on the platform.

## Do / Don't

| ✅ Do | ❌ Don't |
|-------|---------|
| Use on approved background colours | Place on patterned backgrounds |
| Use SVG for web (scalable) | Rasterize below 72 dpi for print |
| Maintain aspect ratio | Stretch or squash the logo |
| Use the gemstone palette | Substitute arbitrary colours |
| Use mascot for friendly/social context | Use mascot for formal documents |
