## Quick Reference

### Colors

| Token | Hex | Use |
|-------|-----|-----|
| Void | `#0D0D1A` | Dark background |
| Supreme Purple | `#7C3AED` | Primary brand color |
| Electric Cyan | `#00E5FF` | AI accent / highlight |
| Sapphire | `#0891B2` | Links / interactive |
| Emerald | `#10B981` | Success states |
| Amber | `#F59E0B` | Warnings |
| Crimson | `#EF4444` | Errors / critical |
| Snow | `#F8F9FF` | Light background |

### Logo Usage

- Use **`logo-dark.svg`** on dark or coloured backgrounds.
- Use **`logo-light.svg`** on white or light backgrounds.
- Use **`icon.svg`** wherever a square/circular icon is needed.
- Maintain clear space equal to the crown mark height on all sides.
- Minimum wordmark width: **160 px**.

### Typography

- **Headings:** Inter 700 / 800
- **Body:** Inter 400 / 500
- **Code:** JetBrains Mono 400 / 500
