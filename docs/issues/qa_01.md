**Labels:** `enhancement`, `testing`, `priority: high`

## User Story

**As a** QA engineer,
**I want** AI to generate test cases from user stories,

## Acceptance Criteria

- [ ] Coverage report shows ≥ 80 % coverage of acceptance criteria
