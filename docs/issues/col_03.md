# [COL-03] AI contextual guidance chatbot for DevOps task onboarding

**Labels:** `enhancement`, `documentation`, `priority: medium`

## User Story

**As a** new team member,
**I want** AI to provide contextual guidance for common DevOps tasks,
**so that** onboarding is faster.

## Acceptance Criteria

- [ ] AI chatbot available in Slack and web UI
- [ ] Answers correctly in ≥ 85 % of common task queries
- [ ] Escalates to human support if unsure (with context forwarded)
- [ ] Onboarding time for new team members tracked and reported
