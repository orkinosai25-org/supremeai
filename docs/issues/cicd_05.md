# [CICD-05] AI-generated release notes from commit history

**Labels:** `enhancement`, `ci/cd`, `priority: medium`

## User Story

**As a** DevOps lead,
**I want** AI to generate release notes from commit history,
**so that** release documentation is always up to date.

## Acceptance Criteria

- [ ] Release notes group commits by type (feat/fix/chore/docs)
- [ ] Generated within 1 minute of merge to main branch
- [ ] Output format is Markdown and posted to the release page
- [ ] Breaking changes are highlighted prominently
