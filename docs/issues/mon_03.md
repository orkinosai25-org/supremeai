# [MON-03] Automated AI incident report generation

**Labels:** `enhancement`, `monitoring`, `priority: medium`

## User Story

**As an** operations engineer,
**I want** AI to generate incident reports automatically,
**so that** post-mortems are more efficient.

## Acceptance Criteria

- [ ] Report generated within 5 minutes of incident resolution
- [ ] Report includes: timeline, impact, root cause, and action items
- [ ] Report is shared to the team Slack channel automatically
- [ ] Action items are automatically created as GitHub issues
