# [DEP-03] AI-generated deployment runbooks per service per release

**Labels:** `enhancement`, `deployment`, `priority: medium`

## User Story

**As a** platform engineer,
**I want** AI to generate deployment runbooks automatically,
**so that** on-call engineers have clear procedures.

## Acceptance Criteria

- [ ] Runbook generated for every service before every release
- [ ] Includes step-by-step deployment and rollback procedures
- [ ] Linked from release notes and deployment dashboard
- [ ] Runbook validated against actual deployment steps in staging
