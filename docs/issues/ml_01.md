# [ML-01] Automated A/B testing for AI model deployments

**Labels:** `enhancement`, `mlops`, `priority: high`

## User Story

**As an** ML engineer,
**I want** to deploy AI models with automated A/B testing,
**so that** model performance is validated before full rollout.

## Acceptance Criteria

- [ ] A/B test configured in < 10 minutes via UI or CLI
- [ ] Winner determined by statistical significance (p < 0.05)
- [ ] Winning model rolled out automatically after significance is reached
- [ ] All experiment metrics logged for audit and reproducibility
- [ ] Supports traffic splits configurable by percentage
