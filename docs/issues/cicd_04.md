# [CICD-04] AI auto-fix for trivial lint and style CI failures

**Labels:** `enhancement`, `ci/cd`, `priority: low`

## User Story

**As a** developer,
**I want** AI to automatically fix trivial lint/style failures in CI,
**so that** I don't block on minor issues.

## Acceptance Criteria

- [ ] Auto-fix applies only to style and lint errors (not logic)
- [ ] PR comment explains every change made
- [ ] No auto-fix applied to logic errors under any circumstance
- [ ] Developer can opt out of auto-fix per repository
