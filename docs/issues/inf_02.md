# [INF-02] AI-powered resource utilisation analysis and auto-scaling

**Labels:** `enhancement`, `infrastructure`, `priority: high`

## User Story

**As a** platform engineer,
**I want** AI to analyse resource utilisation and auto-scale services,
**so that** we optimise costs and performance.

## Acceptance Criteria

- [ ] AI analyses resource utilisation across all services every 5 minutes
- [ ] Scale-up/down recommendations are generated with cost impact estimate
- [ ] Actions can be manually approved or set to auto-apply
- [ ] Monthly cost savings report is generated automatically
- [ ] Scaling events are logged with rationale
