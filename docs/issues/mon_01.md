# [MON-01] AI alert correlation and automated root-cause analysis

**Labels:** `enhancement`, `monitoring`, `priority: high`

## User Story

**As an** SRE,
**I want** AI to correlate alerts and identify root causes automatically,
**so that** MTTR is reduced.

## Acceptance Criteria

- [ ] Correlated root-cause hypothesis surfaced within 60 seconds of alert
- [ ] Accuracy ≥ 75 % measured in retrospective analysis
- [ ] Related alerts are grouped to reduce alert fatigue
- [ ] Suggested remediation steps included with each hypothesis
- [ ] MTTR improvement tracked monthly
