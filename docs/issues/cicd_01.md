# [CICD-01] AI code review and optimisation suggestions on PRs

**Labels:** `enhancement`, `ci/cd`, `priority: high`

## User Story

**As a** developer,
**I want** AI to review my code and suggest optimisations before merging,
**so that** code quality is maintained.

## Acceptance Criteria

- [ ] AI review comments appear on PRs within 2 minutes of opening
- [ ] Issues are categorised as bug, style, or performance
- [ ] False-positive rate is < 5 %
- [ ] Developer can dismiss suggestions with a reason
- [ ] Review metrics (accuracy, adoption) tracked monthly
