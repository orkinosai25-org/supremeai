# [OBS-02] AI performance optimisation suggestions from profiling data

**Labels:** `enhancement`, `observability`, `priority: medium`

## User Story

**As a** developer,
**I want** AI to suggest performance optimisations based on profiling data,
**so that** application performance improves.

## Acceptance Criteria

- [ ] Suggestions reference specific code locations (file + line)
- [ ] Implementing the top suggestion improves p99 latency by ≥ 10 %
- [ ] Suggestions are ranked by estimated impact
- [ ] Before/after benchmarks are run automatically to validate improvements
