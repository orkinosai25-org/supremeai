# [COL-02] AI-generated sprint reports and metrics summaries

**Labels:** `enhancement`, `documentation`, `priority: low`

## User Story

**As a** team member,
**I want** AI to generate sprint reports and metrics summaries,
**so that** stakeholders stay informed.

## Acceptance Criteria

- [ ] Report generated at end of each sprint automatically
- [ ] Includes velocity, blockers, KPI trends, and forecast
- [ ] Report distributed via Slack and email to stakeholders
- [ ] Historical report archive accessible via dashboard
