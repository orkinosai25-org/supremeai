# [MON-02] AI-powered anomaly detection for application metrics

**Labels:** `enhancement`, `monitoring`, `priority: high`

## User Story

**As a** team lead,
**I want** AI-powered anomaly detection for application metrics,
**so that** we catch issues before users are impacted.

## Acceptance Criteria

- [ ] Anomalies detected before SLO breach in ≥ 90 % of incidents
- [ ] Fewer than 2 false positives per day per service
- [ ] Anomaly alerts include context: metric name, baseline, current value, trend
- [ ] Dashboard shows anomaly history per service
