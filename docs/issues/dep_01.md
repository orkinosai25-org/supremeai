# [DEP-01] AI optimal deployment window recommendations

**Labels:** `enhancement`, `deployment`, `priority: medium`

## User Story

**As a** release manager,
**I want** AI to recommend optimal deployment windows based on historical data,
**so that** production incidents are minimised.

## Acceptance Criteria

- [ ] At least 3 deployment windows suggested per release
- [ ] Each window includes: risk score, historical incident rate, and traffic level
- [ ] Recommendations account for team on-call schedules
- [ ] Accuracy of recommendations tracked (incidents in recommended window vs others)
