# [QA-02] AI-powered intelligent test selection for changed code

**Labels:** `enhancement`, `testing`, `priority: medium`

## User Story

**As a** developer,
**I want** AI to predict which tests are likely to fail based on code changes,
**so that** I can run targeted test suites.

## Acceptance Criteria

- [ ] Prediction accuracy ≥ 80 % measured over 30-day window
- [ ] Targeted suite runs ≥ 50 % faster than full suite
- [ ] Report explains why each test was selected
- [ ] Full suite still run on merge to main
