# [OBS-01] AI automated log analysis and actionable insights

**Labels:** `enhancement`, `observability`, `priority: high`

## User Story

**As an** SRE,
**I want** AI to analyse logs and extract actionable insights automatically,
**so that** troubleshooting is faster.

## Acceptance Criteria

- [ ] Insights surfaced within 2 minutes of log ingestion
- [ ] Structured summary includes error patterns, frequency, and trend
- [ ] Top errors linked to known issues or runbooks where available
- [ ] Insights accessible via Slack, dashboard, and API
