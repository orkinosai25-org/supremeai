# [SEC-03] AI-powered vulnerability prioritisation by business risk

**Labels:** `enhancement`, `security`, `priority: high`

## User Story

**As a** DevSecOps engineer,
**I want** AI to prioritise security vulnerabilities by business risk,
**so that** teams focus on the most critical issues.

## Acceptance Criteria

- [ ] Risk score combines CVSS + asset criticality + exploitability
- [ ] Top-10 prioritised vulnerability list updated daily
- [ ] Jira/GitHub tickets auto-created for critical items
- [ ] Teams notified via Slack with context and suggested fix
